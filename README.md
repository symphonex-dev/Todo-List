# 🚀 Todo List React

Une application de gestion d'objectifs quotidiens qui permet d'ajouter, filtrer et suivre ses tâches, avec sauvegarde automatique dans le navigateur, sans backend ni base de données.

## 🔗 Démo en ligne
👉 [Voir le projet en direct](https://todo-list-symphonex.vercel.app)

## ✨ Fonctionnalités clés
* Ajout d'un objectif via un formulaire simple
* Suppression d'un objectif en un clic
* Marquage d'un objectif comme accompli via une checkbox
* Modification du texte d'un objectif par double-clic
* Filtrage des objectifs par statut : tous, en cours, accomplis
* Réinitialisation de tous les objectifs en un clic
* Suppression groupée des objectifs accomplis
* Compteur dynamique des objectifs restants
* Sauvegarde automatique des données dans le localStorage du navigateur

## 🛠️ Tech Stack
* **Frontend :** React 18, Vite 5
* **Backend :** Aucun (données persistées via localStorage)
* **Déploiement :** Vercel

## 💻 Installation & Lancement en local
1. Cloner le dépôt :
   `git clone https://github.com/symphonex-dev/Todo-List.git`
2. Installer les dépendances :
   `npm install`
3. Lancer le projet :
   `npm run dev`
